Interface CacheAdapterInterface

Interface for cache adapters.

Cache adapters allow Guzzle to utilze various frameworks for caching HTTP responses.

The CacheAdapter interface was inspired by the Doctrine 2 ORM:

Direct known implementers

Guzzle\Common\Cache\AbstractCacheAdapter, Guzzle\Common\Cache\ClosureCacheAdapter

Indirect known implementers

Guzzle\Common\Cache\DoctrineCacheAdapter, Guzzle\Common\Cache\NullCacheAdapter, Guzzle\Common\Cache\Zf1CacheAdapter, Guzzle\Common\Cache\Zf2CacheAdapter
Namespace: Guzzle\Common\Cache
